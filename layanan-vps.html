VPS Web Hosting

Apa itu VPS Hosting Server

Layanan web hosting disediakan bagi fakultas/jurusan/lembaga/unit kerja resmi kampus yang mempublikasikan informasinya masing-masing ke internet. Virtual Private Server (VPS) hosting adalah layanan server virtual yang menggunakan teknologi virtualisasi untuk membagi satu server fisik menjadi beberapa server independen. Setiap VPS memiliki resource khusus (CPU, RAM, penyimpanan) yang tidak dibagi dengan pengguna lain, memberikan kontrol penuh, keamanan, dan kinerja lebih stabil.

Siapa sajakah yang bisa menggunakan layanan ini???

  • Fakultas
  • Jurusan
  • Lembaga
  • Direktorat
  • Unit kerja

Dilingkungan Universitas Sriwijaya, untuk kebutuhan publikasi dan kegiatan serta kebutuhan publikasi ejournal dosen.

Fasilitas Layanan

  • Domain

    Nama domain sendiri, dapat menggunakan nama domain sesuai dengan kebutuhan untuk kebutuhan publikasi menggunakan domain *.unsri.ac.id (untuk kebutuhan kegiatan fakultas, jurusan, unit kerja, direktorat) atau *.ejournal.unsri.ac.id (untuk kebutuhan publikasi ejounral dengan OJS) atau *.conf.unsri.ac.id (untuk kebutuhan publikasi acara konferensi).
    Sebagai contoh : fakultas.unsri.ac.id, jurnalejournal.ejournal.unsri.ac.id, konferensi.conf.unsri.ac.id

  • Optimized SSD

    Server hosting menggunakan optimized SSD untuk performa tinggi yang membuat website diakses sangat cepat

  • Control Panel

    Server hosting dilengkapi dengan control panel yang ramah pengguna dengan antarmuka yang sederhana sehingga memudahkan dan membuat website semakin nyaman

  • Quota Disk

    Batasan disk untuk masing-masing domain adalah sebesar 2-10 GB untuk masing-masing pengguna layanan hosting. namun bisa di re-scaling sesuai dengan kebutuhan.

  • Sertifikat SSL

    Mengenkripsi konten website dan membuat pengunjung website mengetahui jika website terpercaya dan aman

  • Backup Mingguan

    Tidak perlu khawatir dengan backup mingguan website. Server hosting memberikan backup mingguan dengan retensi selama 2-3 backup kemudahan restore data hanya dengan 1 klik
